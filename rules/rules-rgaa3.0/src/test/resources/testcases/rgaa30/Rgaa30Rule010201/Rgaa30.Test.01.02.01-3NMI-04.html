Rgaa30 Test.1.2.1 NMI 04

Does each decorative image (img tag), without caption and with an alt attribute, meet the following conditions:
  • The alt attribute has an empty value (alt="")
  • The decorative image does not have a title attribute

NMI : One image is found on the page and is not identified by marker. The other images are either an image link, either an image with a longdesc attribute, either an image marked as informative or an image without alt attribute. They are all out of scope.
